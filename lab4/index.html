¿Qué diferencias y semejanzas hay entre Java y JavaScript?

  • Diferencias: Realmente son lenguajes totalmente diferentes... No tienen nada que ver el uno con el otro.
  • Semejanzas: Si nos ponemos muy técnicos claro que hay semejanzas ya que ambos son lenguajes de programación.

¿Qué métodos tiene el objeto Date? (Menciona al menos 5*)

  • getTime(): Devuelve la cantidad de milisegundos que han transcurrido desde 01/01/1970
  • getDate(): Devuelve el día actual en forma númerica (1 - 31)
  • getMonth(): Devuelve el índice del mes actual (0 - 11)
  • toJSON(): Devuelve el objeto Date() en formato que es compatible con JSON
  • getSeconds(): Devuelve la cantidad de segundos transcurridos desde el último minuto

¿Qué métodos tienen los arreglos? (Menciona al menos 5*)

  • filter(): Como parámetro recibe una función que itera cada elemento del arreglo y si devuelve verdader se guarda en el nuevo arreglo que se va a devolver
  • reduce(): Devuelve un nuevo arreglo reducido con una función que minimiza los elementos del arreglo
  • map(): Corre una función a cada elemento de un arreglo
  • push(): Inserta un elemento al final del arreglo
  • pop(): Remueve el último elemento del arreglo

¿Cómo se declara una variable con alcance local dentro de una función?

Con la palabra clave "var"

Ejemplo
var identifier = ...

¿Qué implicaciones tiene utilizar variables globales dentro de funciones?

Cuando declaras una variable con alcance global no puedes crear una nueva variable en uns subrutina con el mismo nombre ya que el programa no sabrá cuál variable usar. En general, es mala práctica ya que es difícil mantener variables que pueden ser referenciadas en todos lados.
